# DMBC Extras

A lightweight WordPress plugin that adds custom functionality and enhancements for the DMBC site.

## Features

- Adds custom shortcodes and widgets
- Enhances admin screens with additional options
- Provides compatibility tweaks for theme and plugin integrations
- Easy to configure and extend

## Installation

1. Clone or download this repository.
2. Copy the `dmbc-extras` folder to the `wp-content/plugins/` directory of your WordPress installation.
3. Activate the plugin from the WordPress admin dashboard under Plugins.

## Usage

- Visit the Plugins page in WordPress and activate **DMBC Extras**.
- Configure settings as needed from the plugin menu if available.
- Use provided shortcodes or widgets in posts, pages, and theme templates.
